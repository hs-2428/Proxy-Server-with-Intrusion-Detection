# Screenshots

This directory contains screenshots demonstrating the working functionality of the Proxy Server with Intrusion Detection System.

## Available Screenshots:

1. **proxy-settings.png** - Browser proxy configuration showing manual setup with 127.0.0.1:8080
2. **dashboard-overview.png** - Dashboard showing real-time statistics and monitoring
3. **live-alerts.png** - Live alerts panel with multiple XSS detections
4. **blocked-request.png** - Browser showing "Intrusion Detected" message when XSS attack is blocked

## How to Add Screenshots:

Save your screenshots with the following names in this directory:
- `proxy-settings.png`
- `dashboard-overview.png`
- `live-alerts.png`
- `blocked-request.png`

These screenshots are referenced in the main TESTING.md file.
